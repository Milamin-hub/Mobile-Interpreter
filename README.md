# Mobile-Interpreter
The application allows you to run python code on your phone
## Used Technologies
- Python language
- kivy framework 
- kivymd library

# Install project
- git clone https://github.com/Milamin-hub/Mobile-Interpreter.git
- cd mobile_interpreture

# Dependencies Ubuntu
